# IslamIQ Sales Analysis (SQL)



## Overview



This analysis examines sales data from a simulated e-commerce database for IslamIQ, an Islamic education and lifestyle brand. Using SQL, the dataset is explored to understand revenue performance, product profitability, customer acquisition channels, and the impact of discount promotions.



All insights were generated using SQL queries on a relational database.



---



## Database Structure



The dataset models a simplified e-commerce system composed of four relational tables:



**customers**  

Stores customer information including location and signup source.



**products**  

Contains product catalog details such as category, price, and cost.



**orders**  

Represents completed transactions, including order totals, discounts, and status.



**order_items**  

Lists the individual products included in each order.



Table relationships:



customers → orders → order\_items → products



---



## Data Source



Customer data was generated using Mockaroo to simulate realistic

e-commerce customer records. The dataset is included in the repository

as `data/customer_data.csv` and imported into the database using

`LOAD DATA INFILE`.



---



## Key Business Questions



This analysis addresses several common e-commerce questions:



- What is the total revenue generated by the store?

- What is the average order value (AOV)?

- Which products generate the highest gross profit?

- Which products are purchased most frequently?

- Which product categories generate the most revenue and profit?

- How do revenue and profit change over time?

- Which customer signup sources produce the most revenue?

- Which geographic regions generate the strongest sales performance?

- How do discount codes influence revenue and average order value?



---



## Key Insights



- Most revenue comes from full-price orders, indicating strong organic demand without heavy reliance on promotions.

- The **Family Tree of Prophet Muhammad** product line generates the highest overall revenue and profit across its variations.

- **Wall Art** is the highest revenue-generating category, significantly outperforming apparel and other product groups.

- In-person promotions produce the highest average order values, suggesting that live events create higher-intent buyers.

- Online discount codes generate significant transaction volume, though customers using larger discounts appear more price-sensitive.

## SQL Techniques Used



The analysis demonstrates several SQL techniques commonly used in business analytics:



- Multi-table joins

- Aggregations (`SUM`, `AVG`, `COUNT`)

- `GROUP BY` analysis

- `CASE` statements for data categorization

- Window functions for regional performance comparisons

- NULL handling and conditional logic

- Profit and margin calculations
